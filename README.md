# Nexora

Nexora é um projeto de portfólio desenvolvido por **Renan Pires** com o objetivo de demonstrar a construção de um **ecossistema completo de software moderno**.

O sistema simula a gestão de uma loja, incluindo controle de produtos, usuários com diferentes níveis de acesso e operações administrativas, utilizando uma arquitetura composta por múltiplos serviços e aplicações.

Mais do que apenas um sistema funcional, o Nexora foi projetado para demonstrar **boas práticas de engenharia de software, arquitetura escalável e integração entre diferentes tecnologias**.

---

# 🚀 Objetivo do projeto

O Nexora foi criado como um **projeto ambicioso de portfólio**, com foco em demonstrar competências reais de desenvolvimento, incluindo:

- desenvolvimento backend robusto
- arquitetura de sistemas
- construção de APIs
- aplicações web modernas
- aplicações mobile
- integração entre serviços
- boas práticas de engenharia de software

---

# 🧩 Domínio do sistema

O sistema representa um **ambiente de gestão de loja**, contendo funcionalidades como:

- gerenciamento de produtos
- controle de estoque
- cadastro de clientes
- usuários com diferentes níveis de acesso
- hierarquia de funcionários
- operações administrativas

O objetivo é simular desafios comuns encontrados em sistemas de gestão empresarial.

---

# 🏗️ Arquitetura geral

O Nexora é composto por múltiplos componentes que trabalham em conjunto:

### Backend principal
Serviço responsável pela lógica de negócio e APIs do sistema.

Tecnologias:

- Java 21
- Spring Boot
- PostgreSQL

---

### Serviço de dados

Serviço dedicado a análise e manipulação de dados.

Tecnologias:

- Python
- ferramentas de análise de dados
- integração com APIs do sistema

---

### Frontend Web

Interface web moderna para interação com o sistema.

Tecnologias:

- Next.js
- React
- TypeScript

---

### Aplicação Mobile

Aplicação voltada para uso em dispositivos móveis, permitindo acesso às funcionalidades principais do sistema.
